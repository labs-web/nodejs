# Math CLI  

## Description  
Math CLI is a simple command-line tool for performing addition and subtraction.  

## Installation  
1. Clone the repository.  
2. Run `npm install -g` to install globally.  

## Usage  
### CLI  
```bash
math-cli <operation> <num1> <num2>
# Example:
math-cli add 5 3
math-cli subtract 10 4
```
